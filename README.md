# Сборка
```bash
git clone https://github.com/glutenfreedetka/STC_test
cd STC_test
cmake -S . -B build
cd build && cmake --build .
```

# Использование
Для запуска приложения:

```bash
./STC_test <path_to_file>
```
или
```bash
./STC_test -h|--help для вывода справки
```

# Функционал
После успешного открытия и прочтения pcap-файла в консоль будет выведена следующая информация:
````
1) статистика длин пакетов в формате - length %i: counts %i
    1.1) в порядке возрастания длин
    1.2) в порядке возрастания количества
2) статистику для пар МАС-адресов отправитель -> получатель (пара МАС-адресов: количество)
3) количество пакетов по типам протоколов в Ethernet заголовке.
4) результат проверки checksum у IPv4 пакетов в формате - Number of IPv4 packets: %i, with correct checksum: %i

5) выходные файлы в формате [размер_пакета] [пакет_IP] [размер_пакета][пакет_IP]
    5.1)  один файл с пакетами IPv4 (ширина поля "размер_пакета" 2 байта) - при наличии пакетов, с названием "ipv4_%Y%m%d_%H%M%S.bin"
    5.2)  один файл с пакетами IPv6 (ширина поля "размер_пакета" 4 байта) - при наличии пакетов, с названием "ipv6_%Y%m%d_%H%M%S.bin"
    
6) для каждого пакета выполняется расчет контрольной суммы по алгоритму CRC-16-CCITT, рассчитанное добавляется значение в конец пакета,
   полученные пакеты преобразуются в поток кадров HDLC (комбинация 0х7E между пакетами, биты прозрачности) и сохраняются в файл с названием "hdlc_%Y%m%d_%H%M%S.bin"
